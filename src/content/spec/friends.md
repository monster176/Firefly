## 本站信息

申请友链前请先在自己网站添加本站友链，可使用以下信息：

```yaml
站点名称: 次元补给ステーション
站点描述: 把热爱写成文字，让同好在此相遇。
站点链接: https://monster176.github.io/Firefly/
头像链接: https://avatars.githubusercontent.com/u/103022267
```

## 申请友链

请将你的网站信息发送到你的联系方式页面（或通过你设置的评论区/私信渠道联系）。

推荐模板：

```yaml
站点名称: [你的站点名称]
站点描述: [一句话简介]
站点链接: [你的站点链接]
头像链接: [你的头像链接]
```

## 注意事项

1. 互换原则：请先将本站添加到你的友链页。
2. 链接维护：长期无法访问或内容违规的站点会被移除。
3. 网站要求：
   - 内容积极健康，无违法违规信息
   - 无恶意代码、木马、强制弹窗等风险内容
   - 可稳定访问，加载速度合理
   - 有持续更新，不是长期弃更站点
   - 建议支持 HTTPS
